
Platform 1 — GitHub
What it is:
GitHub is a web-based platform used to store, manage, and share code projects using a system called Git version control. It allows developers and students to upload their projects, track changes, and collaborate with others. GitHub is widely used by software companies, open-source communities, and students worldwide.
What I did:
I created my GitHub account using my college email address. I then created a special profile README repository — a repository named exactly the same as my username — which appears on my GitHub profile page. In this README, I added:

Name = Roneet sarode
My branch (Integrated M Tech AI Engineering)
My current year (First Year B.Tech)
My institution (VIT Bhopal University)
One sentence about what I hope to learn

Why I chose it:
GitHub is the most important platform for any Computer Science or engineering student. Recruiters and companies directly check GitHub profiles during internship and job selections. Having a clean GitHub profile from first year gives me a head start.


Platform 2 — LinkedIn
What it is:
LinkedIn is the world's largest professional networking platform with over 900 million users. It works like a digital resume where you can showcase your education, skills, projects, certifications, and work experience. Companies post jobs and internships on LinkedIn, and recruiters directly contact students through it.
What I did:
I created my LinkedIn account and filled in the Education section with:

Degree: Integrated M.Tech
Institution: VIT Bhopal University
Expected Graduation Year: 2030

I kept the profile professional and minimal as instructed.
Why I chose it:
LinkedIn is mandatory for any student who wants internships or placements. It connects me with seniors, professors, professionals, and companies in my field.

Platform 3 — Kaggle
What it is:
Kaggle is an online platform owned by Google, focused on data science, machine learning, and artificial intelligence. It offers free datasets, coding notebooks, competitions, and learning courses. Students can practice real-world data problems and earn certificates and badges.
What I did:
I created a Kaggle account using my college email. I explored the beginner courses available on the platform and set up my profile with my basic details.

For Task 2, I created and set up profiles on three platforms — GitHub, LinkedIn, and Kaggle. Before this task, I had heard of these platforms but never actually created accounts on them. Setting them up made me realize how important a professional digital presence is, even in the first year of college.
GitHub will serve as my technical portfolio where I can store all my coding work over the next four years. LinkedIn will help me build professional connections, follow industry trends, and apply for internships when the time comes. Kaggle will allow me to practice data science skills and earn certificates that add value to my resume.
I learned that creating these profiles early gives students a significant advantage during internship and placement season. Many of my seniors already have strong GitHub and LinkedIn profiles, and I now understand why. My plan is to update these profiles regularly as I learn new skills and complete projects, so that by my final year, they truly represent my abilities and growth as a student.
